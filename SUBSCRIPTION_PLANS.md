# Subscription Plans — ShopLifter (One‑Page)

This one-pager describes the subscription offerings, key features per tier, onboarding flow, and billing guidance for customers and installers.

## Overview
- Software-only subscription for running ShopLifter detection on customer-provided cameras and hardware.
- Edge-first design: local inference for low latency; optional cloud sync for analytics, backups and multi-site views.
- 14–30 day free trial available for new customers.

## Tiers & Features

### Personal — $9 / month
- Up to 2 cameras
- Mobile push notifications
- Local recording (customer hardware)
- Basic cloud sync (alerts metadata)
- Email support

### Standard — $29 / month
- Up to 6 cameras
- Web dashboard & live view
- Alert history (7 days)
- Basic analytics (daily summary)
- Email + chat support

### Business — $99 / month
- Up to 20 cameras
- Multi-user roles & permissions
- Alert history (30 days)
- Priority support (SLA options)
- Optional cloud recording (extra storage cost)

### Enterprise — Custom pricing
- 20+ cameras
- Dedicated onboarding and account manager
- Custom integrations and SLAs
- White-label / reseller options

## Add‑ons
- Additional camera packs (per 10 cameras)
- Cloud storage by GB / month
- Managed provisioning & installation coordination
- Premium support 24/7

## Onboarding Flow (Quick)
1. Customer signs up and creates account (or installer creates account for customer).
2. Provision an edge device or server and flash provided image (or install Docker container).
3. Use provisioning QR or token to bind device to customer account.
4. Discover cameras via network scan or register RTSP/ONVIF URLs.
5. Start detection and configure alert thresholds and notification channels.
6. Trial expires → automatic billing (unless cancelled) or manual upgrade.

## Billing & Trials
- 14–30 day free trial (configurable).
- Monthly billing by default; offer annual billing at ~10% discount.
- Support automatic upgrades/downgrades; prorate billing on mid-cycle changes.

## Reseller / Installer Program
- Volume discounts for installers (e.g., 25% off list price at >10 licenses).
- Reseller portal with device provisioning, billing overview, and customer management.
- Training & certification package for installers.

## Security & Compliance
- Device authentication and encrypted transport (TLS).
- Data retention controls for GDPR/region-specific compliance.
- Option to disable cloud sync if customer requires fully local operation.

## Sales Notes
- Offer quick-start package (preconfigured image + onboarding guide) to reduce install time.
- Use trial conversions and installer partnerships as primary acquisition channels.
